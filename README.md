## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/DRSCrafter/dressify-api.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the API
   ```js
   node API/index.js
   ```
